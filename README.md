## “Be local, Act global”
한국에 있다고, 한국에만 갇혀있을 필요 없어요! Act global  

외국인이라고, 한국에 적응하기 힘드신가요? Be local

> 외국인들과 관심 분야를 통해 서로 교류하는 소셜 네트워킹 서비스, Globud 입니다.
